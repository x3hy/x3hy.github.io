  • [New project] BLUI - 2026-03-12T08:21:45.998Z agoBLUI is a basic api for creating Terminal User Interfaces (TUI). BLUI is incredibly small and robust, BLUI is cross platform and works on Windows, Linux and MacOS. BLUI is compatable with both C and C++. HOW IT WORKS: Inside the header file it has one global variable called _BLUI, this is where all the data for EVERYTHING is held. You can use a function to append a new "element" to the main app structure. Each "element" can have one of two types, it can either be a box or a text "element", text elements can be aligned to the left right or center whereas all "elements" have properties for x and y position on the screen. BLUI doesn't have any input management yet, this is on my to-do list! Feel free to send in a PR or issue!
